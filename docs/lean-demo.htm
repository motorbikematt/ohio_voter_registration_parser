Internal comparison page. This is not the live precincts.info map. It recomputes party lean in the browser to compare formula choices side by side. The published map always uses "Margin over total registration".

Party-lean formula comparison

The choropleth is not winner-take-all. Lean is a signed margin normalised by a denominator, and colour only appears past a threshold. Both the denominator and the threshold are judgement calls -- this page makes them switchable so the difference is visible rather than argued.
